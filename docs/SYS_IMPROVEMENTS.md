## Mejoras Futuras

### Flags Adicionales

```bash
msc sys info --json              # Output en JSON
msc sys info --export report.txt # Exportar a archivo
msc sys info --cpu               # Solo información de CPU
msc sys info --memory            # Solo información de memoria
```

### Comparación de Hardware

```bash
msc sys benchmark   # Ejecutar benchmark rápido
msc sys compare     # Comparar con especificaciones conocidas
```

### Monitoreo en Tiempo Real

```bash
msc sys monitor     # Monitor en tiempo real de CPU, RAM, etc.
```

### Generación de Reportes

```bash
msc sys report --html   # Generar reporte HTML
msc sys report --pdf    # Generar reporte PDF
```
